Processing Instructions
Upload Your Excel File

Upload your daily Excel file containing order information. The file should include columns for Customer Name, Master Ref, ISBN, Title, and other order details.

Database Integration

The application automatically cross-references your upload with the existing book specification database (data.json) to enhance records with additional details like paper specifications, pricing, and technical data.

Automatic Processing

The application will automatically parse your file, decode cover specifications, and transform the data into the ASR Daily Order format with enhanced information from the database.

Review & Download

Review the processed data in the preview table, then download the transformed Excel file ready for production use.
